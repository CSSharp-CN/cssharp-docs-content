# Api 类

## 命名空间
`CounterStrikeSharp.API`

## 说明
提供访问CounterStrikeSharp核心功能的API。

## 公共静态方法

### GetVersion()
获取服务器上运行的CounterStrikeSharp的API版本。

**返回值**: `int` - API版本号

### GetVersionString() 
获取服务器上运行的CounterStrikeSharp的程序集版本,包含git提交哈希。

**返回值**: `string` - 版本字符串

**示例**: `1.0.0+9d8b6be`
