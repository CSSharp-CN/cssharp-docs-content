## 命名空间
`CounterStrikeSharp.API.Core.Attributes.Registration`

## 说明
用于标记控制台命令处理方法。

## 构造函数

### ConsoleCommandAttribute(string command, string description = null)
创建一个新的ConsoleCommandAttribute实例。

**参数**:
- `command`: `string` - 命令名称
- `description`: `string` - 命令描述(可选)

## 属性

### Command
获取命令名称。

**类型**: `string`

### Description
获取命令描述。

**类型**: `string`

## 使用位置
- 方法(可多次使用)
