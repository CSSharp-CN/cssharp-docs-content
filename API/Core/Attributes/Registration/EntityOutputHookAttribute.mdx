# EntityOutputHookAttribute 类

## 命名空间
`CounterStrikeSharp.API.Core.Attributes.Registration`

## 说明
用于标记实体输出钩子处理方法。

## 构造函数

### EntityOutputHookAttribute(string classname, string outputName)
创建一个新的EntityOutputHookAttribute实例。

**参数**:
- `classname`: `string` - 实体类名
- `outputName`: `string` - 输出名称

## 属性

### Classname
获取实体类名。

**类型**: `string`

### OutputName
获取输出名称。

**类型**: `string`

## 使用位置
- 方法(可多次使用)
