# CastFromAttribute 类

## 命名空间
`CounterStrikeSharp.API.Core.Attributes`

## 说明
指示参数应该从ScriptContext中以指定类型获取,然后转换为参数类型。

## 构造函数

### CastFromAttribute(Type type)
创建一个新的CastFromAttribute实例。

**参数**:
- `type`: `Type` - 要从ScriptContext中获取的类型

## 属性

### Type
获取要从ScriptContext中获取的类型。

**类型**: `Type`

- 参数
