# SchemaMemberAttribute 类

## 命名空间
`CounterStrikeSharp.API.Core.Attributes`

## 说明
用于标记Schema成员属性。

## 构造函数

### SchemaMemberAttribute(string className, string memberName)
创建一个新的SchemaMemberAttribute实例。

**参数**:
- `className`: `string` - 类名
- `memberName`: `string` - 成员名称

## 属性

### ClassName
获取类名。

**类型**: `string`

### MemberName
获取成员名称。

**类型**: `string`
