# NetworkedVector 

`NetworkedVector ` 是一个泛型类，用于表示网络同步的向量集合。它实现了 `IReadOnlyCollection ` 接口。

## 属性

### Size
- 类型: `uint`
- 描述: 获取向量的大小。

### Count
- 类型: `int`
- 描述: 获取集合中的元素数量。

### this[int index]
- 类型: `T`
- 描述: 获取指定索引处的元素。
- 限制: 当前仅支持 `CHandle ` 类型的元素。
- 异常: 如果 T 不是 `CHandle ` 类型，会抛出 `NotSupportedException`。

## 方法

### RemoveAll()
```csharp
public void RemoveAll()
```
- 描述: 移除集合中的所有元素。

### GetEnumerator()
```csharp
public IEnumerator  GetEnumerator()
```
- 描述: 返回一个遍历集合的枚举器。

## 备注

这个类主要用于处理需要在网络中同步的向量数据集合。目前仅支持 `CHandle ` 类型的元素。
