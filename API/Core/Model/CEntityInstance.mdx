# CEntityInstance

`CEntityInstance` 是实体实例的基类。它是所有实体类型的最基础类。

## 继承关系

```
CEntityInstance
└── CBaseEntity
    └── CBaseModelEntity
        └── CBasePlayerPawn
            └── CCSPlayerPawn
```

## 备注

这个类是实体系统的基础,所有的实体类型都直接或间接继承自这个类。它提供了实体系统所需的基本功能和属性。
