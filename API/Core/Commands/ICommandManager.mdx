# ICommandManager 接口

## 命名空间
`CounterStrikeSharp.API.Core.Commands`

## 说明
定义命令管理器的接口。

## 方法

### RegisterCommand(CommandDefinition definition)
注册一个命令。

**参数**:
- `definition`: `CommandDefinition` - 要注册的命令定义

### RemoveCommand(CommandDefinition definition)
移除一个命令。

**参数**:
- `definition`: `CommandDefinition` - 要移除的命令定义
