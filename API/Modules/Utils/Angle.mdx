# Angle 类

`Angle` 类提供了游戏中角度计算和操作的功能。

## 命名空间

```csharp
namespace CounterStrikeSharp.API.Modules.Utils
```

## 类定义

```csharp
public class Angle
{
    public float x { get; set; }  // Pitch (俯仰角)
    public float y { get; set; }  // Yaw (偏航角)
    public float z { get; set; }  // Roll (翻滚角)
}
```

## 构造函数

```csharp
public Angle()  // 创建一个所有角度为0的新实例
public Angle(float x, float y, float z)  // 使用指定的x、y、z角度创建新实例
public Angle(Angle angle)  // 从现有角度创建副本
```

## 属性

```csharp
public float Pitch { get => x; set => x = value; }  // 俯仰角
public float Yaw { get => y; set => y = value; }    // 偏航角
public float Roll { get => z; set => z = value; }   // 翻滚角
```

## 主要方法

### 角度操作

```csharp
public void Normalize()  // 标准化角度到 -180 到 180 度范围
public Angle Normalized()  // 返回标准化后的新角度实例
public void Add(Angle other)  // 添加另一个角度
public void Subtract(Angle other)  // 减去另一个角度
public void Scale(float scale)  // 按比例缩放角度
```

### 向量转换

```csharp
public Vector Forward()  // 获取前向向量
public Vector Right()    // 获取右向量
public Vector Up()       // 获取上向量
```

### 运算符重载

```csharp
public static Angle operator +(Angle a, Angle b)  // 角度加法
public static Angle operator -(Angle a, Angle b)  // 角度减法
public static Angle operator *(Angle a, float b)  // 角度乘以标量
public static Angle operator /(Angle a, float b)  // 角度除以标量
```

## 使用示例

```csharp
// 创建一个新角度
var angle = new Angle(30, 45, 0);

// 获取前向向量
var forward = angle.Forward();

// 标准化角度
angle.Normalize();

// 角度运算
var angle2 = new Angle(10, 20, 0);
var sum = angle + angle2;
```

## 说明

- 用于表示和操作3D空间中的角度
- 支持标准的欧拉角表示（Pitch、Yaw、Roll）
- 提供角度标准化功能
- 支持向量转换
- 实现了基本的数学运算
- 适用于视角计算、方向控制等场景
