# AcquireMethod 枚举

`AcquireMethod` 枚举定义了获取资源的不同方法。

## 命名空间

```csharp
namespace CounterStrikeSharp.API.Modules.Utils
```

## 枚举定义

```csharp
public enum AcquireMethod
{
    Find,       // 通过查找获取
    Create      // 通过创建获取
}
```

## 说明

- `Find`: 表示通过查找现有资源的方式获取
- `Create`: 表示通过创建新资源的方式获取
