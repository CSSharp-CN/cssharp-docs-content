## 说明

- 用于高效地表示和比较字符串
- 将字符串转换为整数令牌
- 适用于频繁的字符串比较场景
- 提供了快速的相等性比较
- 节省内存使用
- 线程安全的令牌系统
- 常用于游戏内部的字符串标识
