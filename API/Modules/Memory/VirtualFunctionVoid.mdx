# VirtualFunctionVoid 类

`VirtualFunctionVoid` 类是一个泛型类，用于包装无返回值的虚函数调用。

## 支持的泛型重载

- `VirtualFunctionVoid`：无参数
- `VirtualFunctionVoid<T1>`：1 个参数
- `VirtualFunctionVoid<T1, T2>`：2 个参数
- `VirtualFunctionVoid<T1, T2, T3>`：3 个参数
- `VirtualFunctionVoid<T1, T2, T3, T4>`：4 个参数
- `VirtualFunctionVoid<T1, T2, T3, T4, T5>`：5 个参数
- `VirtualFunctionVoid<T1, T2, T3, T4, T5, T6>`：6 个参数
- `VirtualFunctionVoid<T1, T2, T3, T4, T5, T6, T7>`：7 个参数
- `VirtualFunctionVoid<T1, T2, T3, T4, T5, T6, T7, T8>`：8 个参数
- `VirtualFunctionVoid<T1, T2, T3, T4, T5, T6, T7, T8, T9>`：9 个参数
- `VirtualFunctionVoid<T1, T2, T3, T4, T5, T6, T7, T8, T9, T10>`：10 个参数

## 使用示例

```csharp
// 创建无参数虚函数
var voidFunc = new VirtualFunctionVoid("48 89 5C 24 ?");
voidFunc.Invoke();

// 创建带参数的虚函数
var paramFunc = new VirtualFunctionVoid<int, string>("48 89 5C 24 ?");
paramFunc.Invoke(42, "test");

// 使用二进制路径创建
var binaryFunc = new VirtualFunctionVoid<float>("48 89 5C 24 ?", "server.dll");
binaryFunc.Invoke(3.14f);
```

## 说明

- 提供类型安全的虚函数调用
- 支持基于签名或对象指针的创建
- 自动管理函数指针生命周期
- 支持最多 10 个参数
- 适用于无返回值的虚函数调用
