# DataType 枚举

`DataType` 枚举定义了在内存操作中使用的基本数据类型。

## 命名空间

```csharp
namespace CounterStrikeSharp.API.Modules.Memory
```

## 枚举定义

```csharp
public enum DataType
{
    DATA_TYPE_VOID,
    DATA_TYPE_BOOL,
    DATA_TYPE_CHAR,
    DATA_TYPE_UCHAR,
    DATA_TYPE_SHORT,
    DATA_TYPE_USHORT,
    DATA_TYPE_INT,
    DATA_TYPE_UINT,
    DATA_TYPE_LONG,
    DATA_TYPE_ULONG,
    DATA_TYPE_LONG_LONG,
    DATA_TYPE_ULONG_LONG,
    DATA_TYPE_FLOAT,
    DATA_TYPE_DOUBLE,
    DATA_TYPE_POINTER,
    DATA_TYPE_STRING,
    DATA_TYPE_VARIANT
}
```

## 枚举值说明

- `DATA_TYPE_VOID`: 表示空类型
- `DATA_TYPE_BOOL`: 布尔类型
- `DATA_TYPE_CHAR`: 字符类型
- `DATA_TYPE_UCHAR`: 无符号字符类型
- `DATA_TYPE_SHORT`: 短整型
- `DATA_TYPE_USHORT`: 无符号短整型
- `DATA_TYPE_INT`: 整型
- `DATA_TYPE_UINT`: 无符号整型
- `DATA_TYPE_LONG`: 长整型
- `DATA_TYPE_ULONG`: 无符号���整型
- `DATA_TYPE_LONG_LONG`: 长长整型
- `DATA_TYPE_ULONG_LONG`: 无符号长长整型
- `DATA_TYPE_FLOAT`: 浮点型
- `DATA_TYPE_DOUBLE`: 双精度浮点型
- `DATA_TYPE_POINTER`: 指针类型
- `DATA_TYPE_STRING`: 字符串类型
- `DATA_TYPE_VARIANT`: 变体类型

## DataTypeExtensions 扩展方法

### ToDataType

将 .NET 类型转换为对应的 `DataType`。

```csharp
public static DataType? ToDataType(this Type type)
```

#### 参数

- `type`: 要转换的 .NET 类型

#### 返回值

返回对应的 `DataType`，如果无法转换则返回 `null`。

### ToValidDataType

将 .NET 类型转换为有效的 `DataType`。

```csharp
public static DataType ToValidDataType(this Type type)
```

#### 参数

- `type`: 要转换的 .NET 类型

#### 返回值

返回对应的 `DataType`。

#### 异常

- `NotSupportedException`: 当类型不支持时抛出。

## 类型映射

内置的类型映射关系：

```csharp
{
    { typeof(float), DataType.DATA_TYPE_FLOAT },
    { typeof(IntPtr), DataType.DATA_TYPE_POINTER },
    { typeof(int), DataType.DATA_TYPE_INT },
    { typeof(uint), DataType.DATA_TYPE_UINT },
    { typeof(bool), DataType.DATA_TYPE_BOOL },
    { typeof(string), DataType.DATA_TYPE_STRING },
    { typeof(long), DataType.DATA_TYPE_LONG },
    { typeof(ulong), DataType.DATA_TYPE_ULONG },
    { typeof(short), DataType.DATA_TYPE_SHORT },
    { typeof(sbyte), DataType.DATA_TYPE_UCHAR },
    { typeof(byte), DataType.DATA_TYPE_CHAR }
}
```
