# MemoryFunctionWithReturn

`MemoryFunctionWithReturn` 类系列提供了一组有返回值的内存函数类。

## 描述

这个类系列继承自 `BaseMemoryFunction`,提供了多个泛型版本以支持不同数量的参数和返回值类型。每个类都提供了一个 `Invoke` 方法来调用相应的内存函数并返回结果。

## 类型

### MemoryFunctionWithReturn<TResult>
无参数版本。

### MemoryFunctionWithReturn<T1, TResult>
单参数版本。

### MemoryFunctionWithReturn<T1, T2, TResult>
双参数版本。

### MemoryFunctionWithReturn<T1, T2, T3, TResult>
三参数版本。

### MemoryFunctionWithReturn<T1, T2, T3, T4, TResult>
四参数版本。

### MemoryFunctionWithReturn<T1, T2, T3, T4, T5, TResult>
五参数版本。

### MemoryFunctionWithReturn<T1, T2, T3, T4, T5, T6, TResult>
六参数版本。

### MemoryFunctionWithReturn<T1, T2, T3, T4, T5, T6, T7, TResult>
七参数版本。

### MemoryFunctionWithReturn<T1, T2, T3, T4, T5, T6, T7, T8, TResult>
八参数版本。

### MemoryFunctionWithReturn<T1, T2, T3, T4, T5, T6, T7, T8, T9, TResult>
九参数版本。

### MemoryFunctionWithReturn<T1, T2, T3, T4, T5, T6, T7, T8, T9, T10, TResult>
十参数版本。

## 构造函数

每个类都有两个构造函数:

### (string signature)
使用函数签名创建实例。

- `signature`: 函数签名

### (string signature, string binarypath)
使用函数签名和二进制文件路径创建实例。

- `signature`: 函数签名
- `binarypath`: 二进制文件路径

## 方法

### TResult Invoke([params])
调用内存函数并返回结果。参数数量和类型取决于具体的泛型版本。

- 返回: 函数执行的结果

## 示例

```csharp
// 创建返回整数的无参数内存函数
var func = new MemoryFunctionWithReturn<int>("signature");
int result = func.Invoke();

// 创建返回字符串的双参数内存函数
var func2 = new MemoryFunctionWithReturn<int, bool, string>("signature");
string result2 = func2.Invoke(42, true);
```
