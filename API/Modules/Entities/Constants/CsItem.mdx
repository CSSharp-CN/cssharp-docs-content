# CsItem 枚举

`CsItem` 枚举定义了游戏中所有可用的物品类型，包括武器、装备和其他物品。

## 使用示例

```csharp
public class MyPlugin : BasePlugin
{
    public override void Load(bool hotReload)
    {
        // 给玩家武器
        RegisterEventHandler<EventPlayerSpawn>((ev, info) =>
        {
            var player = ev.Userid;
            
            // 给予基础装备
            GivePlayerItem(player, CsItem.Vest);
            GivePlayerItem(player, CsItem.VestHelmet);
            
            // 根据队伍给予武器
            if (player.Team == CsTeam.CounterTerrorist)
            {
                GivePlayerItem(player, CsItem.M4A4);
                GivePlayerItem(player, CsItem.Usp);
            }
            else
            {
                GivePlayerItem(player, CsItem.Ak47);
                GivePlayerItem(player, CsItem.Glock);
            }
            
            // 给予手榴弹
            GivePlayerItem(player, CsItem.Flashbang);
            GivePlayerItem(player, CsItem.Hegrenade);
            GivePlayerItem(player, CsItem.Smokegrenade);
            
            return HookResult.Continue;
        });
    }
    
    private void GivePlayerItem(CCSPlayerController player, CsItem item)
    {
        if (player.IsValid)
        {
            player.GiveNamedItem(item.ToString());
        }
    }
}
```

## 说明

- 定义所有游戏物品类型
- 用于物品管理和分配
- 支持所有武器类型
- 包含装备和手榴弹
- 用于物品系统
- 支持物品识别
- 便于物品操作
