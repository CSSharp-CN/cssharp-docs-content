# RoundEndReason 枚举

`RoundEndReason` 枚举定义了游戏回合结束的各种可能原因。

## 命名空间

```csharp
namespace CounterStrikeSharp.API.Modules.Entities.Constants
```

## 枚举定义

```csharp
public enum RoundEndReason
```

## 枚举值

### TargetBombed

```csharp
TargetBombed = 1
```

### VIPEscaped

```csharp
VIPEscaped = 2
```


## 使用示例

```csharp
public class MyPlugin : BasePlugin
{
    public override void Load(bool hotReload)
    {
        // 注册回合结束事件处理器
        RegisterEventHandler<EventRoundEnd>((ev, info) =>
        {
            var reason = (RoundEndReason)ev.Reason;
            
            // 根据不同的结束原因处理
            switch (reason)
            {
                case RoundEndReason.TargetBombed:
                    Server.PrintToChatAll("炸弹爆炸！恐怖分子胜利！");
                    break;
                    
                case RoundEndReason.BombDefused:
                    Server.PrintToChatAll("炸弹已拆除！反恐精英胜利！");
                    break;
                    
                case RoundEndReason.CTWin:
                    Server.PrintToChatAll("反恐精英消灭了所有恐怖分子！");
                    break;
                    
                case RoundEndReason.TerroristWin:
                    Server.PrintToChatAll("恐怖分子消灭了所有反恐精英！");
                    break;
                    
                case RoundEndReason.Draw:
                    Server.PrintToChatAll("回合平局！");
                    break;
            }
            
            return HookResult.Continue;
        });
    }
}
```
