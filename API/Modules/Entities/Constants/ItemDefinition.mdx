# ItemDefinition 枚举

## 使用示例

```csharp
// 检查玩家是否持有AWP
if (player.ActiveWeapon.ItemDefinitionIndex == (short)ItemDefinition.AWP)
{
    // 处理AWP相关逻辑
}

// 给玩家一把AK-47
GivePlayerWeapon(player, ItemDefinition.AK_47);

// 检查是否是刀具
bool isKnife = itemDef >= ItemDefinition.BAYONET && itemDef <= ItemDefinition.SKELETON_KNIFE;
```

## 注意事项

- 某些物品ID可能会随游戏更新而改变
- 部分物品可能需要特定条件才能使用
- 纪念品和特殊物品通常有特定的获取条件
- 使用物品时需要考虑游戏平衡性和服务器规则
