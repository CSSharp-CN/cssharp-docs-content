# CounterStrikeSharp API 参考

CounterStrikeSharp是一个用于CS2服务器插件开发的.NET框架。本文档提供了完整的API参考。

## 核心概念

- [Api类](Api.mdx) - 核心API功能
- [BasePlugin类](Core/BasePlugin.mdx) - 插件基类
- [命令系统](Core/Commands/CommandDefinition.mdx) - 命令定义和管理
- [能力系统](Core/Capabilities/Capabilities.mdx) - 插件和玩家能力

## 命名空间

- `CounterStrikeSharp.API` - 核心API
- `CounterStrikeSharp.API.Core` - 核心功能
- `CounterStrikeSharp.API.Core.Attributes` - 特性
- `CounterStrikeSharp.API.Core.Commands` - 命令系统
- `CounterStrikeSharp.API.Core.Capabilities` - 能力系统
